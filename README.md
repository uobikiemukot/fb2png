# fb2png

framebuffer screenshot capture

# feature

-	support 8/15/16/24/32bpp
-	using stb_image_write.h

## usage

 $ fb2png FILE

 $ sleep 5; fb2png FILE

## options

none

## supported image format

-   png by stb_image_write
